Photobox

CSS3 image gallery jQuery plugin

A lightweight image gallery modal window script which uses only CSS3 for silky-smooth animations and transitions. The goal was to create an image gallery script that uses GPU rendering to move things around, instead of the old fashioned javascript way (e.g. most other scripts in existence).

Supports: Firefox, Chrome and IE8+
